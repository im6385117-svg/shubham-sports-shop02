# Shubham Sports Website

Simple storefront plus separate admin panel for product management.

## Files

- `index.html` - customer-facing home page
- `admin.html` - separate admin login and dashboard
- `styles.css` - full site styling
- `script.js` - product storage, filters, login, image upload, and editing logic

## Features

- Separate admin login button on the main page
- Front page opens first, not the login page
- Add products with image upload
- Edit product price
- Change product availability
- Mark products as featured
- Delete products
- Search, category filter, availability filter, and sorting for customers
- Product enquiry button for quick customer contact

## How to Open

Open `index.html` in a browser to use the storefront.

Open `admin.html` to log in as admin and manage products.

## Notes

- Product data is stored in browser `localStorage`.
- Uploaded images are previewed immediately and then shown on the storefront.
